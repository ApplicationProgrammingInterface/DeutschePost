## Rate Limits

All **API** calls are rate limited.

#### Default Limit

Each client can make `15` requests <br>
**per second**, **per API call**.

#### Tracking

Tracking requests are more limited, <br>
allowing only `1` request in `4` hours <br>
for each `barcode` / `awb`.
